Python人工智能学习笔记01-环境搭建

所有操作基于 {视窗操作系统}(Windows)，不能保证Mac和Linux正常操作。
（haha就是测试一下格式）

前言

暑假学校组织了为期三天的数据挖掘/人工智能的培训，有了一点基本的了解，现简单总结一下这几天学习的内容。

Python的安装、环境配置

安装时要勾选环境配置，以便在CMD中运行。
版本要与系统位数相匹配，否则pip不能更新。

Jupyter安装

常用的库安装

  • 更新Pip
  • 安装库

Jupyter

  • 会从默认浏览器启动jupyter notebook
  • 进入后点击右上角的区域new，选择Python3就可以了。
  • M：将代码块变为Markdown文本框
  • B：新建一个代码块
  • 双击D：删除一个代码块

Jupyter也可以用作其他语言的编辑器，并且可以方便地使用一块一块代码编译的方式，很方便。
